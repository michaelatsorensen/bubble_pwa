Signup Flows

1. Normal signup
Auth → minimal onboarding → welcome → home med profil CTA.
2. Event QR signup
Event landing → auth → onboarding → modal: "Gå til event" / "Bliv på forsiden".
3. Personligt QR signup
Profil-preview → auth → onboarding → modal: "Gem kontakt" / "Bliv på forsiden".
4. Netværk invite signup
Auth → onboarding → modal: "Gå til netværk" / "Bliv på forsiden".

Logged-in deep-link tests (v8.17.16)

Tester event-modal flowet for allerede logget-ind brugere. Kræver at du er logget ind på bubbleme.dk i en anden tab.
5. Logged-in event-link (?event=)
Åbner ?event=<id> direkte. Forventet: tids-aware modal med valg (ingen auto-checkin). Brug "Verificer DB" bagefter for at bekræfte at intet skete uden dit valg.
6. Logged-in ?join= på event
Åbner ?join=<event-id>. Forventet: samme tids-aware modal (auto-detect af type). Verificer bagefter.
